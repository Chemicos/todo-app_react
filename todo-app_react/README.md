# React + Vite

Todo web app made with react javascript and tailwind css 
![image](https://github.com/user-attachments/assets/be15a835-902e-46ea-94e3-691a488c7596)

Features: 
 - Add task
 - Complete task
 - Edit task
 - Remove task
 - Translation (EN/RO)
 - Responsive design

# Setup:

# 1. Install dependencies:
`npm install`

# 2. Run the application:
`cd .`
followed by
`npm run dev`
